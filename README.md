
TheriaPy
----------

TheriaPy is a Python tool for computing and visualizing equilibrium assemblages with Theriak-Domino along complex pressure–temperature–composition (P–T–X) paths.

The current version is built on the pytheriak library  (https://github.com/Theriak-Domino/pytheriak).
